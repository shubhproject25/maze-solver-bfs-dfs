# Maze Solver using BFS and DFS

## Problem
Finding a path in a maze is a common problem in navigation and robotics.

## Solution
This project uses BFS and DFS algorithms to find paths in a maze and visualize them.

## Features
- BFS (shortest path)
- DFS traversal
- Animated visualization

## Technologies
- Python
- Matplotlib
- NumPy

## How to Run
pip install -r requirements.txt
python main.py

## Output
- Displays path from start to goal
- Shows difference between BFS and DFS
