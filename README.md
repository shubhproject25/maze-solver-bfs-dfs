# 🧠 Maze Solver using BFS and DFS (AI Project)

## 📌 Overview

This project demonstrates how classic Artificial Intelligence search algorithms — **Breadth-First Search (BFS)** and **Depth-First Search (DFS)** — can be used to solve a maze (pathfinding problem).

The system generates a random maze and visually compares how BFS and DFS find paths from a start point to a goal.

---

## 🎯 Problem Statement

Finding a path from a starting point to a destination in a constrained environment is a common real-world problem in navigation systems, robotics, and game development.

However, understanding how different algorithms solve this problem can be difficult without visualization.

---

## 💡 Solution

This project:

* Generates a random maze
* Applies BFS and DFS algorithms
* Visualizes both solutions side-by-side
* Highlights the difference between optimal and non-optimal paths

---

## ⚙️ Features

* 🔄 Random maze generation (different every run)
* 🔍 BFS (guarantees shortest path)
* 🌊 DFS (explores depth-first, not always optimal)
* 📊 Side-by-side comparison (BFS vs DFS)
* 🎨 Clean UI with colors, grid, and legend
* 📏 Step count display for both algorithms
